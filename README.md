# NWO Vorgehensweisen

Es zielt darauf ab, die Öffentlichkeit über die Methoden und Strategien dieser rechtsextremistischen Bewegung zu informieren, die mit Gruppen wie Anonymous oder QAnon vergleichbar ist.

Die N.W.O. ist die Gruppierung, die alle Schwerstkriminellen vereint, egal ob aus Polizei, Musik, Medien, Politik. Alle Nazis arbeiten hier gemeinsam an einem Ziel!

Darüber hinaus macht das Projekt deutlich, dass die Medien diese terroristische Vereinigung unterstützen, indem sie sie mit süßen oder verharmlosenden Namen wie "Das Cybermobbing Kartell" bezeichnen. Dies dient der Verschleierung der wahren Natur und Aktivitäten der NWO.

Das Ziel ist es, durch objektive Dokumentation und Analyse Bewusstsein zu schaffen und zur Debatte über extremistische Bewegungen beizutragen, ohne selbst Gewalt oder illegale Aktivitäten zu fördern.
